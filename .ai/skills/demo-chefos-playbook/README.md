# Skill — demo-chefos-playbook

## Propósito

Guion operativo para demostrar ChefOS v3 a un cliente potencial (hotel, catering, grupo hostelero).

Cubre:

- setup del entorno de demo (hotel, credenciales, datos sembrados)
- checklist pre-demo
- flujo paso a paso de la demo (40 min)
- plan B si algo falla
- preguntas típicas + respuestas
- cierre + next step

## Cuándo usar

- Cuando hay una demo agendada con cliente.
- Cuando se prepara una demo pública (feria, evento).
- Cuando se onboarda a un nuevo commercial que necesita demo-readiness.

## Cuándo NO usar

- Ni en desarrollo normal.
- Ni para presentaciones internas al equipo (usa `docs/architecture-overview.md`).

## Referencia

Basado en el playbook validado en demo Iago / Eurostars Ourense (abril 2026). Adaptable a cualquier cliente.
