# Skill — production-module-improvements

## Propósito

Banco de mejoras propuestas para el módulo `production` de ChefOS v3, priorizadas por impacto operativo real (cocina profesional) y asignadas a fase MVP 1/2/3.

Es un input explícito para Sprint 07 (production). Las mejoras no se implementan todas — se eligen según el alcance del sprint.

## Cuándo usar

- Al planificar Sprint 07 (production).
- Al revisar Sprint 07 para decidir qué mejora MVP 2 o MVP 3 entra.
- Cuando un stakeholder pide "¿podemos hacer X en producción?" — posiblemente ya está listado aquí con justificación.

## Principio

Producción debe ser:

- **simple y usable** para el personal en móvil
- **potente y controlable** para el jefe de cocina
- sin mezclar responsabilidades con Horarios ni forzar que todo venga de Eventos

## Referencia

Basado en análisis operativo de v2, sesiones con Israel (head chef).
