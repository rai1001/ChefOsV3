Las policies reales viven en las migraciones (`supabase/migrations/`), pero aquí se mantiene una copia legible por módulo para revisión humana.

Helpers canónicos: `is_member_of()`, `get_member_role()`, `check_membership()`. Ver `.ai/specs/database-security.md`.
