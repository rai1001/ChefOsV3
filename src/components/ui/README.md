# src/components/ui/

Componentes UI reutilizables y presentacionales.

Construidos sobre Radix UI primitives + class-variance-authority.

**Permitido**: Button, Input, Select, Dialog, Toast, Badge, Card, Table, Form fields, Skeleton, etc.

**NO permitido**:

- Lógica de negocio
- Acceso a Supabase
- Decisiones específicas de un módulo

Ver `.ai/specs/coding-standards.md § Reglas de componentes` y `.ai/specs/design-tokens.md`.
